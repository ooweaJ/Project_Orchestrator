interface/state 구조와 중앙 Discord 전송

LETHE의 새 오케스트레이션 구조를 오케스트레이터 홈페이지와 Discord 전송 흐름에 반영한 작업 기록입니다.

작업

  • interface/*.html을 홈페이지 iframe의 우선 경로로 적용했습니다.
  • state/*.md를 스냅샷 상태 문서의 우선 경로로 적용했습니다.
  • 날짜별/units HTML 보고서를 개발일지 목록에 표시합니다.
  • 선택 또는 최신 HTML 보고서를 중앙 Discord payload와 첨부로 변환합니다.

검증

  • npm run build 통과.
  • npm run orchestration:dashboard 통과.
  • LETHE interface API와 nested reports API 확인.
  • LETHE Discord dry-run payload와 attachment metadata 확인.
