# Project Brief

## Project

AI Project Orchestrator is a local-first dashboard for managing multiple development projects assisted by Codex or other AI agents.

## User-Facing Goal

- Register local project folders.
- Read Git and file signals without destructive actions.
- Detect risks and likely next tasks.
- Generate useful Codex prompts without requiring paid AI APIs.
- Show the dashboard first from `localhost`.

## Stack

- Frontend: Vite, React, TypeScript
- Backend: Node and Express
- Data: local JSON files under `data/`
- Runtime: local development from this repository

## Portfolio Angle

The project demonstrates practical AI-assisted development operations: safe local scanning, prompt generation, risk detection, portfolio-safe reporting, and reusable orchestration documents.
