# Scope Guard

## Do Not Build Yet

- Do not add paid AI API requirements to the MVP.
- Do not replace local-first behavior with a hosted-only workflow.
- Do not turn the first screen into a landing page.
- Do not add broad automation that can modify registered projects without explicit user action.
- Do not expose private local paths, file names, TODO text, or project names in portfolio mode.
- Do not migrate or delete existing project documentation automatically.

## Default Bias

- Prefer read-only scanning before write actions.
- Prefer scaffold suggestions before modifying another project.
- Prefer compatibility mappings while projects migrate to `docs/orchestration/*`.
