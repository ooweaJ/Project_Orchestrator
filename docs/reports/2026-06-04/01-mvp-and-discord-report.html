MVP 및 Discord 보고서 작업 기록

2026-06-04 기준

어떤 작업

  • 로컬 프로젝트 오케스트레이터의 첫 MVP를 구현했습니다.
  • 사용자용 한글 HTML 보고서를 추가했습니다.
  • Discord 웹훅으로 보고서 요약 embed를 전송하는 기능을 만들었습니다.
  • Discord 전송 시 HTML 보고서 파일도 함께 첨부되도록 확장했습니다.

진행 내용

  • 원격 브랜치와 로컬 커밋이 갈라져 merge 충돌이 발생했고, 현재 MVP 문서 기준으로 해결했습니다.
  • 기본 등록 프로젝트는 UE5 SoulLike와 LETHE Prototype 두 개로 줄였습니다.
  • Discord는 HTML을 본문으로 렌더링하지 않으므로 embed 요약과 HTML 파일 첨부를 함께 쓰는 방식으로 정했습니다.
  • .env의 웹훅 URL은 Git에 올리지 않고, .env.example만 저장했습니다.

결과

  • npm run build가 통과했습니다.
  • npm run report:discord:dry로 embed JSON 생성을 확인했습니다.
  • npm run report:discord로 Discord 전송을 확인했습니다.
  • 이제 보고서는 로컬 HTML 보관용과 Discord 알림용으로 분리되어 운영됩니다.
