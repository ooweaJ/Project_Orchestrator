AI Project Orchestrator 진행 상태

2026-06-08 기준

어떤 작업

  • 홈페이지를 오케스트레이션 문서 중심으로 재구성했습니다.
  • STATUS.md, CURRENT_TASK.md, NEXT_TASKS.md 내용을 대시보드에서 바로 볼 수 있게 했습니다.
  • 기존의 Codex 작업 프롬프트, 위험 신호, 보조 스캔 정보 패널은 기본 화면에서 제거했습니다.
  • Discord 보고는 각 프로젝트가 아니라 오케스트레이터의 중앙 .env에서 보내도록 API를 추가했습니다.

진행 내용

  • 대시보드의 판단 기준을 스캐너 신호에서 오케스트레이션 Markdown 문서로 옮겼습니다.
  • 명령 생성도 현재 상태, 현재 작업, 다음 작업 문서를 기준으로 만들도록 바꿨습니다.
  • Discord 전송은 실제 전송 전에 dry-run 미리보기를 볼 수 있습니다.

결과

  • npm run build가 통과했습니다.
  • GET /api/snapshots에서 오케스트레이션 문서 내용이 내려옵니다.
  • POST /api/projects/lethe-prototype/discord-report dry-run이 통과했습니다.
  • 다음 작업은 실제 Discord 전송 후 보고서 포맷을 다듬는 것입니다.
