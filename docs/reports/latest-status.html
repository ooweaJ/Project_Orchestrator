어떤 작업

  • AI 협업 규칙, 개발 로그, 포트폴리오 기록 구조를 만들었습니다.
  • 로컬 프로젝트 상태를 읽는 첫 MVP 대시보드를 구현했습니다.
  • 기본 등록 프로젝트를 UE5 SoulLike와 LETHE Prototype 두 개로 줄였습니다.
  • README를 한글 사용 설명서로 정리했습니다.

진행 내용

  • 초기 푸시 과정에서 원격 브랜치와 로컬 커밋이 갈라져 충돌이 발생했습니다.
  • 충돌 파일은 현재 MVP 버전을 기준으로 정리했습니다.
  • GitHub에 실제 로컬 경로가 노출되지 않도록 추적 파일에는 예시 경로만 남겼습니다.
  • 브라우저 자동 검증은 Windows sandbox 문제로 실패했지만, HTTP/API 검증으로 대체했습니다.

결과

  • npm run build가 통과했습니다.
  • /api/projects는 UE5 SoulLike와 LETHE Prototype 두 개만 반환합니다.
  • 대시보드는 프로젝트 카드, 상세 패널, 위험도, Codex 프롬프트를 표시할 수 있습니다.
  • 원격 GitHub에는 충돌 해결 커밋까지 푸시되었습니다.
