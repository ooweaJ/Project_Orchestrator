AI Project Orchestrator 진행 상태

2026-06-08 기준

어떤 작업

  • LETHE의 새 오케스트레이션 구조인 interface/와 state/를 홈페이지에서 바로 읽도록 바꿨습니다.
  • LETHE의 HTML 보고서를 Project Orchestrator가 중앙 Discord 웹훅으로 보내는 흐름을 추가했습니다.

진행 내용

  • 대시보드, 다음 지시, 런북 iframe은 docs/orchestration/interface/*.html을 우선합니다.
  • 상태 문서 스캔은 docs/orchestration/state/*.md를 우선합니다.
  • 개발일지는 reports/YYYYMMDD/index.html과 reports/YYYYMMDD/units/*.html까지 재귀로 표시합니다.
  • Discord 전송은 선택된 보고서 또는 최신 unit report를 embed와 HTML 첨부로 변환합니다.

결과

  • npm run build와 npm run orchestration:dashboard가 통과했습니다.
  • LETHE 대시보드 API가 interface/index.html을 반환했습니다.
  • LETHE 스냅샷이 state/*.md 문서를 정상 인식했습니다.
  • LETHE Discord dry-run이 최신 unit HTML 보고서를 embed와 첨부로 변환했습니다.
