AI Project Orchestrator 진행 상태

2026-06-08 기준

어떤 작업

  • docs/orchestration/* Markdown을 읽어 docs/orchestration/index.html을 생성하는 스크립트를 추가했습니다.
  • HTML에는 현재 상태, 이번 작업, 다음 작업, 최근 결정, devlog, reports, 명령, 관련 문서 링크가 카드로 표시됩니다.

진행 내용

  • Markdown은 계속 원본/source of truth로 유지하고, HTML은 사람이 빠르게 보는 generated view로 두었습니다.
  • 현재 프로젝트뿐 아니라 --target과 --all 옵션으로 다른 프로젝트에도 생성할 수 있게 했습니다.

결과

  • npm run orchestration:dashboard가 통과했습니다.
  • npm run orchestration:dashboard -- --all이 통과했습니다.
  • Project_Orchestrator와 LETHE_Prototype에 docs/orchestration/index.html이 생성됐습니다.
  • 다음 작업은 reports/, devlog/, evidence/ 목록 HTML 생성입니다.
