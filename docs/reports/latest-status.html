AI Project Orchestrator 진행 상태

2026-06-08 기준

어떤 작업

  • docs/orchestration/을 재사용 가능한 개인 개발 문서 플러그인으로 정리했습니다.
  • 홈페이지에서 선택 프로젝트에 플러그인 기본 구조를 생성하거나, 기존 프로젝트용 마이그레이션 MD만 생성할 수 있게 했습니다.

진행 내용

  • 설치 스크립트는 interface/ HTML, state/ Markdown, reports/, templates/ 구조를 만듭니다.
  • 개발일지 목록은 날짜별 reports/YYYYMMDD/index.html만 우선 표시합니다.
  • unit HTML 보고서는 Discord 첨부나 상세 링크용으로 계속 사용할 수 있습니다.
  • 마이그레이션/AGENTS/HTML 템플릿은 새 플러그인 개념과 중앙 Discord intake 흐름을 설명하도록 갱신했습니다.

결과

  • 다른 프로젝트에서도 홈페이지 버튼이나 설치 명령으로 동일한 개발 문서 플러그인 구조를 시작할 수 있습니다.
  • LETHE처럼 이미 규약을 적용한 프로젝트는 날짜별 개발일지 중심으로 더 깔끔하게 표시됩니다.
  • Discord 웹훅은 계속 Project Orchestrator 중앙 .env가 담당합니다.
