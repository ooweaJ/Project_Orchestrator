어떤 작업

  • 대시보드의 주요 문구, 버튼, 입력 폼, 위험 뱃지, 추천 작업을 한글화했습니다.
  • blocked 대신 확인 필요로 표시되도록 바꿨습니다.
  • 예시 경로 때문에 스캔이 막힌 경우, 실제 폴더 경로를 등록해야 한다는 안내 박스를 추가했습니다.
  • 위험 메시지와 생성 프롬프트 본문도 한글로 내려오게 바꿨습니다.

진행 내용

  • 사용자가 처음 봤을 때 앱 오류처럼 느껴질 수 있는 blocked 상태를 설명 중심으로 바꿨습니다.
  • 프론트엔드 화면 문구와 백엔드 API 메시지를 함께 바꿔서 영어가 섞이는 부분을 줄였습니다.
  • 브라우저 시각 검증은 Windows sandbox 문제로 열리지 않아 빌드와 API 검증으로 대체했습니다.

결과

  • npm run build가 통과했습니다.
  • GET /api/snapshots가 한글 위험 메시지를 반환했습니다.
  • POST /api/projects/ue5-soullike/prompt가 한글 프롬프트 본문을 반환했습니다.
  • 대시보드는 http://127.0.0.1:5173에서 바로 확인할 수 있습니다.
  • 다음 작업은 실제 LETHE/SoulLike 경로를 연결해서 진짜 프로젝트 상태를 보는 것입니다.
