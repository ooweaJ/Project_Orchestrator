LETHE식 HTML 템플릿과 개발일지

2026-06-08 완료

어떤 작업

  • LETHE의 현재 HTML 형식을 재사용 가능한 오케스트레이션 템플릿으로 문서화했습니다.
  • 홈페이지에서 프로젝트별 HTML 보고서를 개발일지처럼 열람할 수 있게 했습니다.

진행 내용

  • HTML_INTERFACE_TEMPLATE.md에 대시보드, 다음 지시, 런북, 보고서 목록 규칙을 추가했습니다.
  • 마이그레이션 프롬프트가 해당 템플릿을 참조하도록 갱신했습니다.
  • 보고서 API는 docs/orchestration/reports/*.html만 읽고, HTML 제목을 목록에 표시합니다.

결과

  • npm run orchestration:dashboard가 통과했습니다.
  • npm run build가 통과했습니다.
  • LETHE reports 목록과 개별 보고서 API가 200 OK로 응답했습니다.
