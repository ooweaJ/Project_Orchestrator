어떤 작업

  • docs/orchestration/* Markdown을 읽어 index.html 카드 대시보드를 생성하는 스크립트를 추가했습니다.
  • 현재 프로젝트, 단일 대상 프로젝트, 등록된 전체 프로젝트 생성 모드를 지원합니다.

진행 내용

  • Markdown은 원본으로 유지하고 HTML은 generated view로 두는 규칙을 유지했습니다.
  • 현재 상태, 이번 작업, 다음 작업, 최근 결정, devlog, reports, 명령, 문서 링크를 카드로 구성했습니다.

결과

  • npm run orchestration:dashboard 통과.
  • npm run orchestration:dashboard -- --all 통과.
  • Project_Orchestrator와 LETHE_Prototype에 docs/orchestration/index.html이 생성됐습니다.
