홈페이지 Codex CLI 실행

2026-06-08 작업 완료

어떤 작업

  • 홈페이지 명령 칸에서 선택 프로젝트의 Codex CLI를 바로 실행할 수 있게 했습니다.
  • 실행 상태, exit code, 출력, 마지막 Codex 답변을 홈페이지 카드에서 확인할 수 있게 했습니다.

진행 내용

  • POST /api/projects/:id/codex-run으로 Codex 실행을 시작합니다.
  • GET /api/projects/:id/codex-runs와 개별 run API로 최근 실행과 상세 로그를 읽습니다.
  • 각 실행은 docs/orchestration/agent_runs/에 prompt, status, output, final message를 남깁니다.
  • 첫 검증에서 CLI 옵션 위치 문제가 있었고, 최종적으로 approval_policy="never" config override를 사용했습니다.

결과

  • npm run build가 통과했습니다.
  • 검증 run 20260608-095709-1hluj이 complete, exitCode: 0으로 종료됐습니다.
  • 마지막 메시지 codex runner ok가 API 응답에 저장되는 것을 확인했습니다.
