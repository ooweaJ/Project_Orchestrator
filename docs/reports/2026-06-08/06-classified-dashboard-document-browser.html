분류형 대시보드와 개발 문서 브라우저

2026-06-08 완료

어떤 작업

  • 생성 HTML을 판단 흐름에 맞게 다시 분류했습니다.
  • 홈페이지에서 각 프로젝트의 오케스트레이션 문서를 하나씩 열어보는 브라우저를 추가했습니다.

진행 내용

  • 중복 Markdown 카드 영역은 제거하고, 현재 작업/다음 지시는 명령 영역에 합쳤습니다.
  • 생성 HTML은 다음 지시, 현재 상태, 검증/블로커, 기록, 보고서, 운영 명령으로 정리됩니다.
  • 문서 미리보기는 각 프로젝트의 docs/orchestration 내부 파일로 제한했습니다.

결과

  • npm run build가 통과했습니다.
  • npm run orchestration:dashboard -- --all로 LETHE와 Project_Orchestrator HTML을 재생성했습니다.
  • LETHE 문서 목록, 개별 문서, 생성 HTML API가 모두 200 OK로 응답했습니다.
