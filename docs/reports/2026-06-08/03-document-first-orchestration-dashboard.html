문서 중심 오케스트레이션 대시보드

2026-06-08 작업 단위 보고서

어떤 작업

  • 대시보드를 오케스트레이션 Markdown 문서 중심으로 재구성했습니다.
  • 프로젝트별 Discord 보고를 오케스트레이터 중앙 .env 기준으로 보내도록 했습니다.

진행 내용

  • STATUS.md, CURRENT_TASK.md, NEXT_TASKS.md 내용을 화면에서 직접 볼 수 있게 했습니다.
  • 기존 프롬프트/위험/보조 스캔 패널은 기본 화면에서 제거했습니다.
  • Discord 전송은 dry-run 미리보기와 실제 전송으로 나눴습니다.

결과

  • npm run build 통과.
  • GET /api/snapshots에서 오케스트레이션 문서 내용 확인.
  • LETHE Discord 보고 dry-run API 통과.
