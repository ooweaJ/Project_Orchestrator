홈페이지 HTML 대시보드 표시

2026-06-08 완료

어떤 작업

  • 선택한 프로젝트의 생성된 오케스트레이션 HTML을 홈페이지 안에서 볼 수 있게 했습니다.
  • 상세 화면에 HTML 대시보드 패널과 새 창 링크를 추가했습니다.

진행 내용

  • 서버가 등록 프로젝트의 docs/orchestration/index.html을 읽어 HTML로 반환합니다.
  • 홈페이지는 이 HTML을 iframe으로 표시하므로 LETHE 같은 진행 중 프로젝트의 카드 뷰를 바로 확인할 수 있습니다.

결과

  • npm run build가 통과했습니다.
  • LETHE 대시보드 API가 200 OK로 HTML을 반환했습니다.
  • 홈페이지 dev 서버가 200 OK로 응답했습니다.
