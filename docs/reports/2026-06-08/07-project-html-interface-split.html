프로젝트 HTML 인터페이스 분리

2026-06-08 완료

어떤 작업

  • 프로젝트별 HTML 인터페이스를 index.html, command.html, runbook.html로 나눴습니다.
  • 홈페이지는 이 세 HTML을 읽어 프로젝트 대시보드, 다음 지시, 런북으로 표시합니다.

진행 내용

  • index.html은 프로젝트 대시보드 역할만 합니다.
  • command.html은 명령 프롬프트 위에 놓이는 다음 지시 블럭입니다.
  • runbook.html은 설명이 붙은 운영 절차 블럭입니다.
  • 위험 배지, 보고 블럭, 문서 브라우저는 메인 운영 흐름에서 제거했습니다.

결과

  • npm run build가 통과했습니다.
  • npm run orchestration:dashboard -- --all이 HTML 세트를 생성했습니다.
  • LETHE 프로젝트 대시보드, 다음 지시, 런북 API가 모두 200 OK로 응답했습니다.
